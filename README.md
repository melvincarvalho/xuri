## Introduction

xuri, executable URIs

## Methods

- npx (nmp)
- file (exectuable files)
- ssh (executable remote files)
- npub (exectuable nostr npubs)
- cgi-bin (HTTP POST)
- docker (deploy)

### Method npm

```bash
xuri:npm:<package> [args]
```

### Method file

```bash
xuri:file:<path> [args]
```

### Method ssh

```bash
xuri:ssh:<path> [args]
```


### Method ssh

```bash
xuri:npub:<npub> [args]
```


### Method http

```bash
xuri:http:<url> [POST]
```

### Method docker

```bash
xuri:docker:<dockerid> 
```
